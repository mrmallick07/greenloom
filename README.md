## 🌟 Features
- **Premium Aesthetics:** Forest greens, earthy sand tones, and elegant typography to reflect sustainable luxury.
- **Product Categories Matrix:** Showcases 6 distinct catalogs (Shopping Bags, Wine Carriers, Totes, Planters, Conference Office Supplies, and Gift Bags).
- **Responsive Design:** Fluid layouts that look flawless acting as a full catalog on desktop or a quick-reference site on mobile.
- **Micro-Interactions:** Smooth GSAP-inspired scroll animations, dynamic hover states, and interactive stat counters.
- **Lead Generation:** Optimized contact forms specifically routing wholesale and global B2B queries efficiently.
- **AI-Generated Assets:** All product catalog imagery was custom generated to maintain a unified, high-end studio lighting aesthetic.

## 🛠️ Technology Stack
- **HTML5:** Semantic architecture
- **CSS3:** Custom properties (variables), modern Grid/Flexbox layouts, backdrop filters, and custom keyframe animations 
- **JavaScript (ES6):** Intersection Observers for lazy-loaded scroll animations and dynamic number counters

## 🚀 Live Preview (GitHub Pages)
Once deployed via GitHub pages, the website resides at:
`https://mrmallick07.github.io/greenloom/`

**To enable GitHub pages:**
1. Go to your repository **Settings** tab.
2. Click on **Pages** in the left sidebar.
3. Under **Build and deployment**, set Source to **Deploy from a branch**.
4. Select the **main** branch and save. Your site will be live within 2-3 minutes!
